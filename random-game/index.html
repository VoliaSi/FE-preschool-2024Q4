Bulls and Cows

|
Result

You need to guess a 4-digit number.

The number cannot begin with 0.

All the digits of the number must be different.

If you guess a digit of the number on its own place, you'll get B (bull) in the result.

If you guess a digit of the number, but not on its own place, you'll get C (cow) in the result.

The game ends when you guess the number = get B B B B in the result.

A HINT FOR THE REVIEWERS!!! The guessed number is logged to the console.
